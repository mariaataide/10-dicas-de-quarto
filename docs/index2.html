10 Fonte Awesome

Existe uma extensão para colocar o Fonte Awesome dentro do Quarto. Para instalar, usar o seguinte código no terminal :

quarto install extension quarto-ext/fontawesome

Esse código pode ser encontrado no GitHub.

Tabela 1: Deflatores da PNAD Contínual Anual
Deflator Descrição
CO1 Deflator, a preços médios do próprio ano, utilizado para variáveis de rendimento habitual.
CO1e Deflator, a preços médios do próprio ano, utilizado para variáveis de rendimento efetivo.
CO2 Deflator, a preços médios do último ano, utilizado para variáveis de rendimento habitual.
CO2e Deflator, a preços médios do último ano, utilizado para variáveis de rendimento efetivo.
CO3 Deflator, a preços médios do último ano, utilizado para a linha de U$ 1,90 do indicador 1.1.1 dos Objetivos de Desenvolvimento Sustentável (ODS).
